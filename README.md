# 🔐 Cyber Security Toolkit

## 📌 Project Overview
The Cyber Security Toolkit is a Python-based project designed for basic network security analysis. It includes multiple tools used in ethical hacking and cybersecurity testing such as port scanning, network scanning, banner grabbing, and vulnerability checking.

This project is developed for academic purposes to understand how cybersecurity tools work in real-world environments.

---

## ⚙️ Features

- 🔍 Port Scanner – Scans open ports on a target system 
- 🌐 Network Scanner – Detects active devices in a network 
- 📡 Banner Grabber – Extracts service information from ports 
- ⚠️ Vulnerability Checker – Identifies possible risky ports 

---

## 🛠️ Technologies Used

- Python 3 
- Socket Programming 
- Nmap (for scanning) 
- Kali Linux / Windows
